一 、docker 部署 music-tag 音乐刮削

music-tag 音乐刮削_后台地址：http://localhost:8001

1 、终端命令创建docker-compose.yml文件

mkdir -p /mnt/mydisk/home/music-tag && touch /mnt/mydisk/home/music-tag/docker-compose.yml && cat > /mnt/mydisk/home/music-tag/docker-compose.yml <<'EOF'
services:
   music-tag:
      image: xhongc/music_tag_web:latest
      container_name: music-tag
      restart: unless-stopped
      ports:
         - 8001:8001
      volumes:
         - ./music:/app/media:rw
         - ./config:/app/data

## 默认用户名/密码：admin/admin
EOF

cd /mnt/mydisk/home/music-tag && docker-compose up -d

2 、拉取并运行 

1

cd /mnt/mydisk/home/music-tag && docker-compose up -d

3 、停止并删除 

1

cd /mnt/mydisk/home/music-tag && docker-compose down

4 、拉取镜像 

1

docker pull xhongc/music_tag_web:latest

5 、容器升级 

1

cd /mnt/mydisk/home/music-tag && docker-compose down && docker-compose pull && docker-compose up -d && docker image prune -f
